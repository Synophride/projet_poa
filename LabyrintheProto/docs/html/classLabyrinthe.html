Informe sur l'état d'occupation de la case (i, j).

WALL si la case est un mur, BOX si la case est occupée par une boîte, GARDE si la case est occupée par un garde, JOUEUR si la case est occupée par le joueur, TREASURE si la case est occupée par le trésor
